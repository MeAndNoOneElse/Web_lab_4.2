# frontendVUE — Vue 3 фронтенд для web_lab_4

Переписанный фронтенд на Vue 3 + Vite + TypeScript.

## Установка и запуск

```bash
cd frontendVUE
npm install

# Запуск с реальным Java-бэкендом (порт 8080)
npm run dev

# Запуск с mock-сервером (порт 3000)
npm run dev:mock
```

Приложение откроется на `http://localhost:5173`

## Структура проекта

```
src/
├── main.ts               — точка входа
├── App.vue               — корневой компонент
├── router/index.ts       — маршруты с guard-ами
├── models/types.ts       — TypeScript интерфейсы
├── services/
│   ├── api.ts            — Axios инстанс + refresh interceptor
│   ├── authService.ts    — login, register, logout, токены
│   └── pointService.ts   — работа с точками
├── views/
│   ├── LoginView.vue     — страница входа
│   ├── RegisterView.vue  — страница регистрации
│   └── MainView.vue      — главная страница
├── components/
│   ├── GraphCanvas.vue   — SVG-граф с зонами и точками
│   ├── PointForm.vue     — форма ввода координат
│   └── ResultsTable.vue  — таблица результатов
└── assets/styles.css     — глобальные стили (только светлая тема)
```

## Функционал

### Страница входа / регистрации
- Валидация полей (минимальная длина логина/пароля, формат email, совпадение паролей)
- Отображение ошибок под каждым полем
- Показ сообщений об ошибках сервера

### Главная страница
- **SVG-граф**: три зоны попадания (прямоугольник, четверть круга, треугольник), оси со стрелками, засечки, точки (зелёные = попадание, красные = промах). Клик по графу отправляет точку.
- **Форма ввода**: radio для X ∈ {-3..5}, текстовое поле для Y ∈ [-3, 3], radio для R ∈ {1..5}
- **Таблица**: X, Y, R, Результат, Время выполнения, Время создания
- **Адаптивный layout**: Desktop (≥1164px) — три колонки, Tablet (≥710px) — две, Mobile — одна
- **Refresh-токены**: автоматически обновляются при 401 ответе
- Без контроля сессий, только светлая тема

## Переключение между бэкендами

Через переменную окружения:
```bash
VITE_API_TARGET=http://localhost:3000 npm run dev
```

Или:
```bash
npm run dev:mock   # → mock-сервер на :3000
npm run dev        # → Java-бэкенд на :8080
```
